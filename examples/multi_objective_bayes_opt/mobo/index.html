    Multi-objective Bayesian Optimization

    TNK function $n=2$ variables: $x_i \in [0, \pi], i=1,2$

    Objectives:

    • $f_i(x) = x_i$

    Constraints:

    • $g_1(x) = -x_1^2 -x_2^2 + 1 + 0.1 \cos\left(16 \arctan \frac{x_1}{x_2}\right) \le 0$
    • $g_2(x) = (x_1 - 1/2)^2 + (x_2-1/2)^2 \le 0.5$
